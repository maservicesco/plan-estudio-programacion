# 📗 Extensiones recomendadas de VS Code

> Cómo instalar: icono de **Extensions** en la barra izquierda (o `Ctrl+Shift+X`) → buscá el nombre → **Install**.
> Para asegurarte de instalar la correcta, verificá el **publisher (autor)** que pongo entre paréntesis.

⭐ = instalá ahora · 🔸 = cuando lleguemos a esa fase · 🎨 = comodidad/opcional

---

## ⭐ ESENCIALES (instalar ahora — te ayudan desde el día 1)

| Extensión | Publisher | Para qué sirve |
|-----------|-----------|----------------|
| ⭐ **Prettier - Code formatter** | esbenp | Ordena y formatea tu código automáticamente al guardar. Te enseña buen estilo sin pensarlo. |
| ⭐ **Error Lens** | usernamehw | Muestra los errores **en la misma línea**, en rojo, al instante. Buenísima para *ver* tus errores mientras escribís. |
| ⭐ **ESLint** | dbaeumer | Detecta errores y malas prácticas en JavaScript mientras tipeás. Como un profesor que te corrige en vivo. |
| ⭐ **Code Spell Checker** | streetsidesoftware | Te avisa de typos (errores de tipeo) en tu código y comentarios. Evita bugs tontos. |
| ⭐ **Live Server** | ritwickdey | Abre tus páginas HTML en el navegador y las **recarga solas** al guardar. Clave para la Fase 2 (frontend). |

### Configurar Prettier para que formatee al guardar (recomendado)
1. `Ctrl + ,` (abre Settings) → buscá **"Format On Save"** → tildá la casilla.
2. Buscá **"Default Formatter"** → elegí **Prettier**.
Ahora cada vez que guardás (`Ctrl+S`), tu código se ordena solo. 🪄

## ⚠️ Consejo de profesor sobre la IA que escribe código (Copilot y similares)
Existen extensiones de IA que **autocompletan/escriben el código por vos** (GitHub Copilot, etc.).

> **Mi recomendación mientras aprendés los fundamentos: mantenelas apagadas o no las instales todavía.**

¿Por qué? Porque en esta etapa el objetivo es que **tu cabeza** arme la lógica. Si la IA te completa las líneas, aprendés a *aceptar* código en vez de a *pensarlo* — justo el músculo que queremos entrenar. Usá la IA (como a mí) para **entender y preguntar**, no para que teclee por vos. Más adelante, cuando los fundamentos estén sólidos, esas herramientas SÍ te van a acelerar sin frenar tu aprendizaje.
