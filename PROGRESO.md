### Evaluación inicial — 2026-07-01
**Nivel: Principiante falso ("false beginner").** Tiene exposición real e informal (HTML puro hace años, algo de PHP, SaaS/webs con ayuda de IA) pero los modelos mentales core están difusos o levemente errados. No es cero real.

Diagnóstico por pregunta:
- **Cliente-servidor:** ⚠️ Parcial. Identifica que él escribe el dominio, pero confunde "cliente" (es el navegador, no él) y no menciona el servidor; "Enter = camarero" incorrecto (el camarero es la red).
- **Variable:** ⚠️ Intuición de "varía" correcta, pero la define como "acción" en vez de "contenedor/caja que guarda un valor".
- **HTML/CSS/JS:** ⚠️ HTML y CSS bien; error clave: cree que JS "es el lenguaje en que se escribe" el resto. JS = comportamiento/interactividad.
- **Terminal:** ⚠️ Cómodo usándola (buena señal), pero la confunde con DevTools/inspeccionar de Chrome. Sabe que cmd/VSCode son terminal.
- **Git/GitHub:** ✗ Concepto errado (cree que valida código/busca errores/quizá BD). No distingue Git de GitHub. → Reforzar en Fase 0.
- **Background:** HTML estructural puro, algo de PHP, reemplaza bloques con IA. Entiende "construir por partes".

**Foco Fase 0-1:** solidificar modelos mentales correctos (variable, rol de JS, Git/GitHub, cliente-servidor). Aprende rápido y tiene intuición; el trabajo es precisión conceptual, no partir de cero absoluto.
